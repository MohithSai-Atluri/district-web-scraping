# district-web-scraping
By using selenium through java to interact with district by zomato to get some available events for this weekend and also to find available languages for movies for the specified city.

Since district is a Single Page Application(SPA), each the script to interact with each component is written in it's own java file and all locators and common methods are in Constants file.

An invalid mobile number is given and a screen shot is taken and saved!
